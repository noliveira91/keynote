keyNOTE

App que nos permite guardar registos de varias contas, exemplo conta de um forum: Regista e grava nome utilzador, email , password.

Primeiro ecra tem icons de varios sites, ao carregar em cada icon abre um novo ecra onde tera la essa informação, pode ser possivel editar, ou apagar.Dados são guardados localmente.

Para entra na app e necesario conta de cliente com email e password.Dados para o login sao guardados no servidor.

FINAL :

Projeto e uma app que guarda password e registos de contas, do tipo nome, email e password da conta do Facebook, com o intuito de ter uma base de dados com todas as passwords.

A app regista a na base de dados local todos os dados, e vai buscar a internet ao servidor "ibizaclubpt.com" a um ficheiro .xml as sugestões para Auto-Complete do nome de cada conta.

É possível criar, editar e eliminar registos.
